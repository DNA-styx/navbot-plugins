# Zombie Panic! Source Objective Support

Plugin implements objective completion logic for bots.    

## ConVars

- sm_navbot_zpo_support_detection_radius: Detection radius for *search* objective types.
- sm_navbot_zpo_support_debug: If enabled, logs debug messages.

## Supported Map List

|Map Name|Status|Humans Required|
|:---:|:---:|:---:|
|Biotec (zpo_biotec)|Partially implemented.|Yes.|

Any map not listed above are not supported.    
If **Humans Required** is *Yes*, human players are required to complete one or more objectives.
